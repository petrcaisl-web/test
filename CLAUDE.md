# IAIA Feedback App
## Kontext
Toto je interní aplikace pro sběr zpětné vazby na tým IAIA (AI competence center)
v Air Bank. Tým má 3 streamy:
- Stream 1: AI v byznysových produktech
- Stream 2: AI empowerment pro ne-IT uživatele
- Stream 3: AI akcelerace pro IT/vývoj
## Tech stack
- Jednoduchá HTML/CSS/JS aplikace (single page)
- Žádný backend framework — data se ukládají do JSON souboru nebo localStorage
- Český jazyk pro UI
- Moderní, čistý design
## Konvence
- Kód komentovat česky
- Responzivní design
- Přístupnost (WCAG základy)
